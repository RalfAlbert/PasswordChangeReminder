Einstellungen für die Passwort Änderungs-Erinnerung

Haupteinstellungen

Intervall

Stellen Sie den Intervall ein nachdem ein Passwort spätestens geändert werden soll. Der Interval kann in Tagen, Wochen, Monaten und Jahren eingegeben werden.
Doch beachten Sie bitte dass das Plugin die Einheiten "Wochen", "Monate" und "Jahre" intern in Tagen umrechnet. So werden z.B. 3 Monate intern in 90 Tage umgerechnet, Monate mit 31 oder 28 Tagen werden also nicht berücksichtigt. Diese Umrechnung führt dazu das z.B. 4 Wochen (4x7 Tage = 28 Tage) nicht das gleiche sind wie 1 Monat (30 Tage). Dies kann dazu führen das die Erinnerung einige Tage früher oder später angezeigt wird als erwartet. Eine genauere Umrechnung wird ggf. in einer späteren Version umgesetzt.

Die Umrechnung erfolgt mit folgenden Werten:

  • Woche: 7 Tage (Bsp.: 3 Wochen entsprechen 21 Tage)
  • Monat: 30 Tage (Bsp.: 3 Monate entsprechen 90 Tage)
  • Jahr: 365 Tage (Bsp.: 3 Jahre entsprechen 1095 Tage)

Zusätzliche Einstellungen

Hinweis ignorieren

Benutzer können den Hinweis zur Passwortänderung normalerweise ausblenden (ignorieren). Möchten Sie dies nicht und den Benutzer quasi dazu zwingen sein Passwort zu ändern, so können Sie diese Funktion hier deaktivieren.

Hinweis wieder einblenden nach...

Sie können hier einen Zeitraum festlegen nach dem der Hinweis zur Änderung des Passwortes wieder angezeigt werden soll nachdem er vom Benutzer ausgeblendet wurde. Geben Sie den Zeitraum in Stunden und Minuten ein.
Sollten Sie einen ungültigen Wert eingeben, so wird der Zeitraum auf den Standardwert zurück gesetzt.

Zusätzliche Nachricht

Hier können Sie eine Zusätzliche Nachricht eingeben die dem Benutzer im Hinweis angezeigt wird. So können Sie auf verbindliche Regeln hinweisen oder eine ausführliche Begründung eingeben warum es wichtig ist regelmäßig das Passwort zu ändern.
Möchten Sie keinen zusätzlichen Hinweis ausgeben lassen, so lassen sie das Feld einfach frei.

Zur Formatierung des Textes können Sie HTML verwenden.
